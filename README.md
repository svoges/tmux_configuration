# tmux_configuration

Moving around tmux windows
ctrl + a + vim commands to move around

resizing a window left to right
ctrl + a + > [many times to move]

split vertically
ctrl + a + v

split horizontally
ctrl + a + s

detach from tmux session
ctrl + a + d

creating a new pane
ctrl + a + N

moving to different panes
ctrl + a + n (next) OR ctrl + a + p (previous)
